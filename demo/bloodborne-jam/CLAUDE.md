# Bloodborne Jam — Unity 6 Project

Local 1v1 PvP fighter prototype. Gothic atmosphere, 3-hour jam scope.

## Entry Point

```
Director_Ship("SAMPLE_GDD.md")
```

## Kurallar

- 2D URP projesi — tüm fizik Rigidbody2D
- Tüm UI: UIDocument + UXML/USS — UGUI Canvas yasak
- Tüm sahne mutasyonu: CRAFT ops
- Gameplay kodu → B19 (unity-developer, gpt-5.4)
- Input: 2 ayrı InputActionAsset (dual keyboard, same machine)
- Audio: horror.json mixer preset
- PostFX: horror.json postfx preset
